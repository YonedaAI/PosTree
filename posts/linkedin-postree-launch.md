Introducing PosTree — an open-source CLI that lets you write social media posts as markdown files and publish to 10+ platforms from your terminal.

Why? Because managing social media for research projects shouldn't require a browser tab for every platform.

PosTree is built for developers and researchers who want:
- Content as code (markdown + YAML frontmatter)
- Git-trackable publishing history
- LLM-powered content generation (Claude, GPT-4, Gemini)
- Scheduled publishing via Claude Code triggers
- One OAuth backend (Postiz) for all platforms

Currently supports: Twitter/X, LinkedIn, Mastodon, Bluesky, Dev.to, Medium, Facebook, Reddit, Discord, and Discourse.

Built by YonedaAI Research Collective.
